  • Job boards — LinkedIn, Indeed, Glassdoor, RemoteOK & more

Getting your LinkedIn cookie (30 sec, desktop browser only):
  1. Open linkedin.com and make sure you're logged in
  2. Right-click anywhere → Inspect (or press Ctrl+Shift+I / Cmd+Option+I)
  3. Go to Application → Cookies → click the LinkedIn URL
  4. Find li_at — double-click its Value and copy it
  5. Paste the value into the field above (after li_at=, which is already filled in)
Getting your Indeed cookie (30 sec, desktop browser only):
  1. Open indeed.com and make sure you're logged in
  2. Right-click anywhere → Inspect (or press Ctrl+Shift+I / Cmd+Option+I)
  3. Go to Application → Cookies → click the Indeed URL
  4. Find CTK — double-click its Value and copy it
  5. Paste the value into the field above (after CTK=, which is already filled in)
Getting your Glassdoor cookie (30 sec, desktop browser only):
  1. Open glassdoor.com and make sure you're logged in
  2. Right-click anywhere → Inspect (or press Ctrl+Shift+I / Cmd+Option+I)
  3. Go to Application → Cookies → click the Glassdoor URL
  4. Find GSESSIONID — double-click its Value and copy it
  5. Paste the value into the field above (after GSESSIONID=, which is already filled in)
Getting your session cookie (30 sec, desktop browser only):
  1. Open the job site and make sure you're logged in
  2. Right-click anywhere → Inspect (or press Ctrl+Shift+I / Cmd+Option+I)
  3. Go to Application → Cookies → click the site URL
  4. Find the main session cookie, copy its Value, and paste as cookie_name=VALUE
⚠️ This step requires a desktop browser — DevTools aren't available on phones.

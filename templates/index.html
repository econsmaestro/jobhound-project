🔓 Hitting a login wall? Add your session cookie
Getting your LinkedIn cookie (30 sec, desktop browser only):
  1. Open linkedin.com, make sure you're logged in
  2. Press F12 → Application tab → Cookies → click the LinkedIn URL
  3. Find li_at — double-click its Value and copy it
  4. Paste it above as: li_at=PASTE_HERE
Getting your Indeed cookie (30 sec, desktop browser only):
  1. Open indeed.com, make sure you're logged in
  2. Press F12 → Application → Cookies → click the Indeed URL
  3. Find CTK — copy its Value
  4. Paste it above as: CTK=PASTE_HERE
Getting your Glassdoor cookie (30 sec, desktop browser only):
  1. Open glassdoor.com, make sure you're logged in
  2. Press F12 → Application → Cookies → click the Glassdoor URL
  3. Find GSESSIONID — copy its Value
  4. Paste it above as: GSESSIONID=PASTE_HERE
Getting your session cookie (30 sec, desktop browser only):
  1. Open the job site and make sure you're logged in
  2. Press F12 → Application → Cookies → click the site URL
  3. Find the main session cookie and copy its Value
  4. Paste it above as: cookie_name=VALUE
⚠️ This step requires a desktop browser — DevTools aren't available on phones.
